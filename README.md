## 🚀 Key Project Highlights

* **Agile Management**: Fully tracked via Jira using Sprints and Backlogs.
* **Modular Architecture**: Organized into 6 independent Epic modules (Order, Product, User, Inventory, Feedback, and Delivery) for high scalability.
* **Role-Based Access Control (RBAC)**: Personalized dashboards for Admin, Manager, Staff, Inventory roles, and Customers.
* **Production-Ready Security**: Password hashing with `bcryptjs` and 4-digit security PINs for administrative actions.

## 🛠️ Tech Stack

* **Frontend**: React.js, Tailwind CSS, Vite.
* **Backend**: Node.js, Express.js.
* **Database**: MongoDB Atlas.
* **Media**: Local image uploads via Multer.
